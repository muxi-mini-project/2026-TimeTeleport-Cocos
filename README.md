## 游戏简介

### 故事背景

主角是都市中一名从事机械重复工作的普通职员——或许是在流水线上重复拧动螺丝，或许是在办公桌前机械整理文件，亦或是在监控室里盯着永不变化的画面。

日复一日的无聊工作耗尽了生活的色彩。就在又一个令人昏昏欲睡的工作日午后，疲惫的他突然走神，脑海中竟不受控制地浮现出一片奇幻场景。

**双重场景视觉体验**：

- **右下角**：现实无聊工作场景（低饱和度、低对比度）
- **游戏主界面**：脑海中的奇幻画面（2D像素风平台跳跃游戏）

这正是他潜意识中为逃离无聊而构建的**精神避难所**。

### 核心特色

- **双维度时空跳跃**：可在"过去/未来"两个时空之间自由切换
- **双重场景体验**：动态工作场景 + 奇幻游戏界面的视觉反差
- **流畅操作手感**：参考《蔚蓝》《超级马里奥》的经典平台跳跃操作
- **特色道具系统**：钩爪、无人机、护盾等丰富道具
- **成就收集**：时空碎片、未来晶片、远古化石等收集要素

## 核心玩法

### 基础规则

玩家需在**时空能量耗尽倒计时**结束前，操控角色从关卡起点（随机时空锚点）抵达终点（核心时空节点）。

> ⚠️ 倒计时归零前未抵达则判定为"时空湮灭"，需重新挑战

### 时空切换

游戏包含**两个时空**，玩家可通过两种方式切换：

#### 方式一：手动切换

- 按 `Shift` 键切换同一空间的不同时空（过去 ↔ 未来）

#### 方式二：传送旋涡

- 角色靠近**传送旋涡**时，时间和空间均会切换

## 游戏机制

### 障碍物

#### 现代时空障碍

- **现代激光**：从墙面发射的激光，有一定间歇时间，需跳跃躲避

#### 过去时空障碍

- **远古地火**：道路中间的火焰，需跳跃躲避
- **恐龙**：在固定区域左右移动的敌人

#### 双重时空障碍

- **古老碎石/现代桥梁**：在碎石上停留3秒后碎石会碎掉
- **三合一滚动球**：热球 / 冰球 / 泥球
- **保安**：在现代时空固定区域左右移动的敌人

### 地图特色

| 特色       | 说明            |
| -------- | ------------- |
| **传送旋涡** | 靠近后切换至另一个时空   |
| **藤蔓云梯** | 使用固定位置的云梯爬向高处 |
| **望远镜**  | 预知后续地图        |

## 道具系统

### 工具道具

| 道具          | 效果                          |
| ----------- | --------------------------- |
| 🛡️ **防御衣** | 防御三种球（热/冰/泥）的伤害             |
| 📦 **宝箱**   | 可获得任意一个道具（偏稀有）              |
| 🪝 **钩爪**   | 快速移动到地图上固定的勾点（勾点有动画提示）      |
| 🚁 **无人机**  | 到达仅跳跃无法达到的平台，使用时间固定，方向由玩家控制 |
| 🔰 **护盾**   | 碰到敌人时获得较高的跳跃高度              |

## 收集系统

### 收集物

| 收集物           | 类型     | 说明                  |
| ------------- | ------ | ------------------- |
| 💎 **时空碎片**   | 通用     | 散落在地图各角落，收集完全即可完美通关 |

| 🦕 **远古化石**   | 过去时空特有 | 过去地图专属收集物           |

### 成就系统

- **常规成就**：收集普通收集物、特殊收集物、彩蛋收集物
- **特殊成就**：完成特殊操作达成
